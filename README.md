Python Arena Game

A terminal-based arena combat game written in Python.  
The project focuses on Object-Oriented Programming (OOP) concepts and basic game mechanics.

---

Features

- Multiple character types:
- Warrior (Armor system)
- Monster (Bonus damage to weak enemies)
- Mage (Mana-based attacks)
- Inheritance and method overriding
- Randomized combat system
- File handling for winner history
- Simple menu-driven gameplay

---

Concepts Used

- Classes & Objects
- Inheritance
- Polymorphism
- `@classmethod` and `@staticmethod`
- File I/O operations
- Random module
- Exception handling

---

How to Run

Make sure you have **Python 3.x** installed.

```bash
python-arena.py
